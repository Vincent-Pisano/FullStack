Mini tutoriel Angular

Angular est un framework utilisant le TypeScript.

Il est basé sur les modules/composants.

Ils permettent la création de SPA ou Single page Application en chargeant la page par section et plus toute entière.

Le chargement de la page se fait sur le navigateur Web et plus sur le serveur comme les applications. traditionnelle

Cela libère le serveur pour répondre plus rapidement aux demandes du client.

Différences

TypeScript et Javascript
  1. TypeScript doit être compilé
  2. TypeScript englobe Javascript
  3. TypeScript utilise des types et des interfaces
  4. TypeScript à une meilleur structure de code
  5. Type script offre plus de controle

Mini tutoriel CRUD

Les CRUD sont les opérations que font les API ou Application Programming Interface.

Les API les plus utilisées sont les REST API ou Representational State Transfert API.

Les REST API sont responsables des services ou conversation entre le client et la base de données.

En programmation Web, le format le plus utilisé dans cette conversation est le JSON.

Mini tutoriel Relations Objets

La programmation orienté objet nécessite des relations entre les classes qui la composent.

Elles définissent comment les classes intéragissent entre elles.

Relations Objets

Nom et définition
  1. Association: relation sémantique entre deux classes
  2. Multiplicité: relation One To Many / Many To Many
  3. Aggrégation: relation de contenance faible (Association de type "contient")
  4. Composition: aggrégation plus forte (perte d'intégrité de l'objet sans cette relation)
  5. Réalisation: relation entre une interface et une classe pour garantir la présence de certaine méthodes
  6. Héritage: relation parent-enfant (l'enfant obtient les variables et méthodes (public/protected) du parents
